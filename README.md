# EntreLanas - Backend (REST API)

Welcome to the **EntreLanas** backend repository. This is an e-commerce application designed for the management and sale of handmade crafts, crochet items, and haberdashery supplies.

This project implements a robust **RESTFUL API** using **Java and Spring Boot**, following a professional layered architecture (Controller, Service, Repository, DTO, Mapper) with data persistence in **MySQL**.

## Tech Stack

* **Language:** Java 17 (or higher)
* **Framework:** Spring Boot 3.x
* **Dependency Manager:** Maven
* **Database:** MySQL
* **Persistence:** Spring Data JPA / Hibernate
* **IDE:** Visual Studio Code (with "Extension Pack for Java") or IntelliJ IDEA

---

## Prerequisites

To run this application locally, you need to have the following installed:

1.  **Java JDK 17+**: [Download here](https://www.oracle.com/java/technologies/downloads/).
2.  **MySQL Server**: It is recommended to use **WAMP** (which includes MySQL and Apache) or install MySQL Server separately.
3.  **Git**: To clone the repository.
4.  **IDE**: Visual Studio Code or IntelliJ IDEA.
